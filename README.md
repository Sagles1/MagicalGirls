# Magical Girls Mod - Forge 1.12.2

Un mod Forge 1.12.2 pour RLcraft ajoutant des compagnons Magical Girls.

## 🎀 Fonctionnalités Actuelles

### Soraka Companion
- **Item**: Soul Gem (clic droit pour invoquer/téléporter)
- **Stats**: 20 HP, Vitesse: 0.35, Dégâts: 2.0
- **Comportement**: Suit le joueur, soigne, attaque les monstres

## Installation

Téléchargez le JAR depuis [Releases](../../releases) et placez-le dans `mods/`

## Compilation

./gradlew build
